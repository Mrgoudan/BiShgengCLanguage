# A - Keywords

The following list contains the keywords newly added by BiSheng C along with descriptions of their functionality.

- `_Trait`: Used to define a `_Trait`
- `_Impl`: Used to implement one or more `_Trait`s for a type
- `This`: Used in a member function to denote the current type
- `this`: Used in an instance member function to denote the current instance or a pointer to the current instance
- `constexpr`: Used to define a compile-time constant expression or a function evaluated at compile time
- `_Safe`: Used to declare a safe zone
- `_Unsafe`: Used to declare a non-safe zone
- `_Owned`: Used to qualify a pointer type that owns its value, or an `_Owned struct`
- `_Borrow`: Used to qualify a borrow pointer type
- `_Const`: Used to obtain an immutable borrow
- `_Mut`: Used to obtain a mutable borrow
- `_Private`: Used to qualify private members of an `_Owned struct`
- `_Public`: Used to qualify public members of an `_Owned struct`
- `_Nonnull`: Used to qualify a non-null pointer type
- `_Nullable`: Used to qualify a nullable pointer type
- `nullptr`: The literal used to represent a null pointer
- `_Async`: Used to qualify an async function
- `_Await`: Used for async function calls
